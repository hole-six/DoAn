# Ghi chú rà soát robustness

- Tổng thể 25 sơ đồ robustness: **khớp nghiệp vụ, có thể du di**.
- Không có sơ đồ nào bắt buộc phải sửa gấp để khớp luồng code hiện tại.
- Một vài sơ đồ dùng tên boundary kiểu `form-*` nhưng UI thật đã chuyển sang `drawer/detail`:
  - `10-robustness-ung-tuyen-tin-tuyen-dung.puml`
  - `12-robustness-quan-ly-lich-phong-van.puml`
  - `17-robustness-quan-ly-ho-so-ung-vien.puml`
  - `18-robustness-moi-ung-vien-phong-van.puml`
  - `19-robustness-cap-nhat-ket-qua-phong-van.puml`
- Nếu muốn đồng bộ tài liệu hơn nữa, ưu tiên sửa tên boundary trong các sơ đồ trên; còn lại giữ nguyên được.
