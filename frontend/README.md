# Frontend - React TypeScript

## Chay du an

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

## Cau truc entry hien tai

- `src/khoitao.tsx`: diem vao frontend.
- `src/ungdung.tsx`: root app.
- `src/nenchung.css`: global style.
- `src/ungdung.css`: style theo man hinh.

## Quy tac bat buoc

- Quy tac naming + design dung **duy nhat** tai lieu goc: `../DESIGN.md`.
- Ten file `.ts`/`.tsx` dat lien, khong dau gach ngang.
- Ten ham va bien trong code dung tieng Viet khong dau.
